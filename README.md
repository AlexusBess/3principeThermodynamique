# 📘 Les principes de la thermodynamique — HTML Only

Mini-site pédagogique présentant les principes fondamentaux de la thermodynamique, réalisé entièrement en **HTML**, sans CSS ni JavaScript.

---

## 🎯 Objectif du projet

Ce projet a été réalisé dans une démarche d’apprentissage et de consolidation des bases du développement web, avec un focus particulier sur :

- La **structure sémantique HTML5**
- L’accessibilité (ARIA, navigation interne, skip-link)
- L’intégration d’expressions mathématiques via **MathML**
- L’organisation logique d’un contenu scientifique

L’objectif était de produire un document structuré, lisible et rigoureux uniquement avec les outils natifs du HTML.

---

## 📚 Contenu

Le site présente :

- 🔹 Le **principe zéro** (fondement de la température)
- 🔹 Le **premier principe** (conservation de l’énergie)
- 🔹 Le **deuxième principe** (entropie et irréversibilité)
- 🔹 Le **troisième principe** (limite du zéro absolu)

Chaque section comprend :
- Un énoncé
- Une formulation mathématique
- Une explication conceptuelle
- Une application physique

---

## 🛠 Technologies utilisées

- HTML5 sémantique
- MathML
- ARIA (accessibilité)
- Navigation interne via ancres

Aucune dépendance externe.

---

## 💡 Choix techniques

- Utilisation de `<section>`, `<article>`, `<nav>`, `<header>`, `<main>`
- Liaison `aria-labelledby`
- Skip link pour navigation clavier
- Utilisation correcte des fonctions d’état et notations scientifiques en MathML
- Structuration progressive du contenu (pédagogie)
